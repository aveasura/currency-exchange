# Currency Exchange
[ТЗ проекта](https://zhukovsd.github.io/java-backend-learning-course/projects/currency-exchange/)

REST API для работы с валютами и обменными курсами.

Для быстрого тестирования API в проект добавлены файлы:
- [`positive-requests.http`](./positive-requests.http) - проверка позитивных сценариев
- [`negative-requests.http`](./negative-requests.http) - проверка негативных сценариев

## Стек
- Java 21
- Jakarta Servlet API
- JDBC
- SQLite
- Maven
- HTML
- 
## Интерфейс
- HTML
- CSS
- JavaScript

## Функциональность
- получение списка валют
- получение валюты по коду
- добавление новой валюты
- получение списка обменных курсов
- получение курса по валютной паре
- добавление нового обменного курса
- обновление существующего обменного курса
- конвертация суммы из одной валюты в другую
